Información General OVI

El OVI trabajado se basa en la utilziación de GIT, GITHUB, HTML 5 y CSS3, en el cual se explica los conceptos sobre estas herramientas.

Objetivos Ovi

1. Crear un maquetado inicial, con lo conceptos básicos de HTML 5, CSS 3 y otros conceptos adicionales, que ayudaran en el desarrollo de la página web.

2. Aprender a utilizar y manejar la estructura básica de HTML, para crear un esquema de una página web, también aprender a utilizar bibliotecas externas como lo es Bootstrap y adaptarlo a nuestro HTML.

3. Presentar los conceptos básicos de GIT, GITHUB, HTML 5 y CSS 3.

4. Implementar efectos a las páginas web por medio del CSS, utilizado efectos hover, active, before, after, etc.

Objetivos Curso

5. Introducción, profundización e información complementaria sobre el uso de GIT y GITHUB para gestionar proyectos colaborativos de diseño web, HTML5 y CSS3 para creación de contenidos en formato web (Paginas y Sitios Web)

Tematica del Curso

La tematica que se utilizo para el desarrollo de esta actividad fue Diseños de Sitios Web (301122).

Información Adicional

En el desarrollo de esta actividad se conoce mas afondo el funcionamiento de GitHub, GitHubPages, Css, HTML y otras implementaciones adicionales que se hicieron en esta pagina, esta cuenta con secciones de Slider(Carrucel) para la visualizzción de los videos.
