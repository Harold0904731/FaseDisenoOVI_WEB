GITHUB

El desarrollo de software se vuelve más fácil cuando tiene acceso a las herramientas que necesita para hacer su mejor trabajo. Con flujos de trabajo simples y coherentes, puede reducir la fatiga del cambio de contexto, agilizar los procesos de autenticación y enfocarse en los problemas que realmente importan. Para ayudarlo a obtener una experiencia de desarrollador perfecta, continuamos nuestra asociación con Microsoft para llevar Azure Pipelines a GitHub. Este nuevo servicio CI / CD le permite construir, probar e implementar continuamente en cualquier plataforma, y ​​es gratis para repositorios de código abierto.

Automáticamente active su canalización con cambios en su repositorio cuando conecte Azure Pipelines con GitHub. Obtenga informes de estado completos, código anotado e información detallada, todo dentro de la interfaz de GitHub.
